# Portal de Treinamento Ampernet Telecom 🚀

---

> Uma aplicação web interativa e gamificada desenvolvida para a capacitação inicial de técnicos e colaboradores operacionais da Ampernet Telecom em fluxos de comunicação intersetorial, requisições automatizadas e procedimentos operacionais.

## 💻 Sobre o Projeto

O **Portal de Treinamento** foi desenhado exclusivamente para preparar a equipe técnica no uso de ferramentas automatizadas via Telegram e fluxos de comunicação interna. O sistema conta com:
* **Identificação Obrigatória:** Registro do técnico para fins de certificação.
* **Simuladores Interativos:** Testes práticos em tempo real simulando a experiência com os bots corporativos.
* **Gamificação:** Acúmulo de pontos (XP) e barra de progresso conforme a conclusão dos módulos.
* **Avaliação Final:** Quiz de proficiência com liberação automática de certificado em PDF (mínimo de 80% de aproveitamento).

---

## 📚 Módulos de Capacitação Incluídos

1. **Amper Interno Bot (`@amper_opa_bot`):** Simulação do fluxo de triagem e geração de Protocolo de Atendimento.
2. **SuperAmpbot (`@SuperAmpbot`):** Fluxo estruturado para requisição de ferramentas e insumos de estoque.
3. **CST & Indicações:** Procedimentos operacionais para reporte de Sinal Degradado e registro de indicações comerciais.
